# Task 2: Exploratory Data Analysis (EDA)

## Objective
To understand the Titanic dataset using statistical analysis and data visualizations.

## Steps Performed
- Loaded the Titanic dataset using Pandas.
- Explored the dataset structure and checked missing values.
- Generated summary statistics for numerical features.
- Created a histogram to analyze data distribution.
- Created a boxplot to identify outliers.
- Generated a pairplot to visualize relationships between features.
- Created a correlation heatmap to study feature correlations.
- Derived basic insights from the dataset using statistics and visualizations.
